# Тестовое задание для МайКедр

### Задание

```
Вам нужно создать страницу со списком пользователей. 
На странице должны быть представлены имя, 
фамилия и электронная почта каждого пользователя. 
Каждый из столбцов можно отсортировать, нажав на стрелочки в таблице. 
Также сверху есть поиск, который позволяет найти пользователя по имени, 
фамилии или почте.

Требования
Дизайн страницы должен быть простым и чистым.
Список пользователей должен загружаться с помощью API, 
предоставленного сервисом jsonplaceholder 
(https://jsonplaceholder.typicode.com/)
Для создания страницы следует использовать React или Vue.js.
Код должен быть хорошо организован и понятен.

Оценка
Оцениваться будут следующие аспекты:
– Качество кода (читабельность, организация, документация).
– Качество пользовательского интерфейса (дизайн, удобство использования).
– Соответствие требованиям.

Сдача задания
Код должен быть выложен на GitHub. Ссылка на репозиторий с заданием должна быть 
отправлена на электронную почту. В письме необходимо указать среднее время, за которое вы ожидаете выполнить задание.
```
